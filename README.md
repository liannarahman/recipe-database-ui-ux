## Overview
This project is a UI UX design for a recipe database webpage created in Figma. The goal was to design an intuitive and visually clear interface that allows users to browse, search, and view recipes efficiently.

## Project Focus
- User centered design and layout
- Clear navigation and content hierarchy
- Accessibility and readability
- Designing for real world use cases
